Static block:
	Static block is executed only once when the class is loaded.
	Static block does not get executed during instantiation
	Static block does not belong to an object

Static variable:
	Static variable belongs to class not to object
        
Static method:
	Static method belongs to class not to object.
	Static method can call only another static method
	Static method can access only static variable
        Static method can be called from another a non-static method
